# Culturify

(PL) Aplikacja do zarządzania wydarzeniami oraz biletami online. Napisana w języku React/TypeScript (Frontend) oraz PHP (backend) wraz z relacyjną bazą danych SQL. Aplikacja powstała przez rozbudowę i modyfikację aplikacji eDoktor także mojego wykonania. Aplikacja posiada:
- logowanie użytkownika, organizatora i admina;
- możliwość zakupu biletów na określone wydarzenia;
- panel admina z możliwością dodawania i usuwania organizatora;
- panel użytkownika z posiadanymi biletami oraz danymi osobowymi i doładowywanym portfelem;
- panel organizaotra z informacjami o jego wydarzeniach;
- rynek Culturify umozliwiający odkupienie biletu od innej osoby oraz sprzedaż swojego własnego za równowartość zakupionego biletu;
- kilka innych mniej lub bardziej użytecznych podstron (łącznie 16 widoków).
Data wykonania projektu: 01.05.2024 - 18.06.2024.

(EN) An application for managing events and tickets online. Written in React/TypeScript (frontend) and PHP (backend) with a relational SQL database. The application was created by extending and modifying the eDoktor application, also developed by me. The application features:
- login for users, organizers, and admins;
- the ability to purchase tickets for specific events;
- an admin panel with the ability to add and remove organizers;
- a user panel with their tickets, personal information, and a top-up wallet;
- an organizer panel with information about their events;
- the Culturify marketplace, allowing you to buy tickets from others and sell your own for the equivalent of the purchased ticket;
- several other subpages (16 views in total).

Project completion date: May 1, 2024 - June 18, 2024.
